 # raid
## RAID 0
- 优点：使用 n 颗硬盘，即可拥有将近 n 倍的读写效能。
- 缺点：数据安全性较低，同组数组中任一硬盘发生问题就会造成数据遗失。
- 硬盘数量：最少 2 个。
